# MyGetWell Mobile App - UI Overview

## Application Structure

### Home Screen
The main screen of the MyGetWell mobile app features:

**Header:**
- App Bar with "MyGetWell" title
- Light blue color scheme (Material Design 3)

**Body:**
- Centered layout with vertical column
- Health icon (health_and_safety) - 100px size in primary blue color
- Welcome message: "Welcome to MyGetWell" (24pt, bold)
- Tagline: "Your health and wellness companion" (16pt, grey)
- "Get Started" button (elevated button with primary color)

**Interaction:**
- Tapping "Get Started" shows a snackbar with welcome message
- Material Design 3 animations and transitions

## Color Scheme
- Primary Color: Blue (from Material seed color)
- Background: White
- Text: Dark grey/Black
- Accent: Blue tones

## Supported Platforms
- ✅ Android (API 21+)
- ✅ iOS (iOS 11+)
- ✅ Web (Progressive Web App)
