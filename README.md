# Customer-Mobile-App
A Flutter standalone mobile app for https://mygetwell.app

## Description
This is the customer mobile application for MyGetWell, a health and wellness platform. The app provides users with easy access to health services and wellness resources.

## Getting Started

### Prerequisites
- Flutter SDK (3.0.0 or higher)
- Android Studio or Xcode for mobile development
- Dart SDK

### Installation
1. Clone the repository
```bash
git clone https://github.com/sbasharjr/Customer-Mobile-App.git
cd Customer-Mobile-App
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```

## Features
- Clean and modern UI design
- Material Design 3 implementation
- Support for Android and iOS platforms
- Health and wellness focused interface

## Testing
Run tests with:
```bash
flutter test
```

## Building
### Android
```bash
flutter build apk
```

### iOS
```bash
flutter build ios
```
